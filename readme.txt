== Description ==

Katari User Role Editor is a powerful tool for managing WordPress user roles and capabilities. It provides an intuitive interface for creating, editing, and deleting user roles, as well as managing their capabilities. Perfect for site administrators who need fine-grained control over user permissions.

**Key Features:**

* Create and customize new user roles
* Edit existing WordPress roles safely
* Delete unnecessary roles without affecting users
* Manage role capabilities with a user-friendly interface
* Clone existing roles to create new ones
* Bulk assign or remove capabilities
* Import/Export role configurations
* Secure and efficient permission management

**Use Cases:**

* Create custom roles for different team members
* Set up contributor roles with specific permissions
* Manage multiple client access levels
* Create hierarchical permission structures

**Security Note:**
The plugin includes safety measures to prevent accidental removal of critical capabilities and roles.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Access the plugin settings through the WordPress admin menu

== Frequently Asked Questions ==

= Is it safe to edit user roles? =

Yes, but make sure you understand the implications of changing user roles and capabilities as it can affect site security and user access.

= Can I restore default WordPress roles? =

Yes, the plugin includes a feature to reset roles to WordPress defaults if needed.

= Will users be affected if I delete a role? =

Users assigned to a deleted role will be automatically assigned to the default subscriber role to ensure they maintain basic access.
