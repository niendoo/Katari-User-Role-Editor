A comprehensive WordPress user role and capability management plugin.

== Description ==

Katari User Role Editor is a powerful tool for managing WordPress user roles and capabilities. It provides an intuitive interface for creating, editing, and deleting user roles, as well as managing their capabilities.

Features:
* Create new user roles
* Edit existing roles
* Delete unnecessary roles
* Manage role capabilities
* User-friendly interface
* Secure and efficient

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Access the plugin settings through the WordPress admin menu

== Frequently Asked Questions ==

= Is it safe to edit user roles? =

Yes, but make sure you understand the implications of changing user roles and capabilities as it can affect site security and user access.
